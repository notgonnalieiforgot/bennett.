# Stack & Tools MOC

Map of content for the studio's technology stack — the frameworks, data layers, automation, motion, hosting, and UI tools used across every client build. Each tool note links back here and out to the [[Projects MOC]] projects that use it.

## Framework & UI
- [[Next.js 16]] — primary app framework (App Router, React 19, Turbopack).
- [[Tailwind v4]] — CSS-first styling and design tokens.
- [[shadcn ui]] — Radix + Tailwind component layer.

## Data
- [[Supabase]] — Postgres + Auth + Storage + RLS for ops dashboards.
- [[Neon Postgres]] — serverless Postgres via Vercel storage.
- [[Drizzle ORM]] — typed schema/queries over Postgres.

## Email & Automation
- [[Resend]] — transactional email (lead notify + auto-reply).
- [[n8n]] — webhook automation for content + lead pipelines.

## Motion & Assets
- [[GSAP Lenis Framer Motion]] — the scroll/motion stack.
- [[Higgsfield]] — AI image/video asset generation (MCP).

## Hosting
- [[Vercel]] — primary host/deploys for Next.js apps.
- [[Cloudflare Pages]] — static hosting + Pages Functions.
