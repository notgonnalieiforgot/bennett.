# Scroll Motion - Higgsfield Exploded

End-to-end workflow for scroll-driven exploded product views (GSAP ScrollTrigger + react-three-fiber or stacked image layers) paired with Higgsfield-generated assets curated to the site's product imagery. Reach for it when building scrollytelling UI, exploded vial/product sections, amino-acid visual layers, or anything mentioning Higgsfield, Seedance, `whileTap`, ScrollTrigger, or R3F exploded views.

## What it does
- Builds scroll-scrubbed exploded product sections — components fly apart and reassemble as the user scrolls — using GSAP ScrollTrigger with either 3D (R3F) or layered 2D image planes.
- Adds Framer Motion tap/hover micro-interactions (`whileTap`, hover states) to product hotspots and buttons.
- Generates and curates supporting visuals via the Higgsfield MCP, matched to the product photography on the site.
- Requires Claude Code with the claude.ai HIGGSFIELD connector for image gen; Cursor agents without that MCP pull assets via manifest using the user's Higgsfield credits in Claude Code.
