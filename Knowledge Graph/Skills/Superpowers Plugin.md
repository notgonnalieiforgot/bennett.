# Superpowers Plugin

A plugin of process-discipline skills that govern *how* work gets done rather than what gets built — brainstorming, test-driven development, systematic debugging, writing plans, dispatching parallel agents, and verification before completion. Reach for it to bring rigor and repeatability to any non-trivial build.

## What it does
- **Brainstorming** — structured idea generation and option framing before committing to an approach.
- **Test-Driven Development** — write tests first, then implementation, for any feature or bugfix.
- **Systematic Debugging** — disciplined hypothesis-driven debugging instead of guess-and-check.
- **Writing Plans / Executing Plans** — produce written implementation plans and execute them with review checkpoints.
- **Dispatching Parallel Agents / Subagent-Driven Development** — fan out 2+ independent tasks with no shared state.
- **Verification Before Completion** and **finishing-a-development-branch** — confirm the work actually works and choose merge/PR/cleanup cleanly.
