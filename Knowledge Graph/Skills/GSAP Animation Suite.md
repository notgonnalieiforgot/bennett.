# GSAP Animation Suite

The full GSAP skill cluster covers everything from core tweens to scroll-linked orchestration, plugin ecosystem, framework integration, and production-grade performance. Reach for it any time scroll animation, pinning, sequencing, or 60fps motion engineering is on the table — it replaces ad-hoc CSS transitions and lighter motion libraries for anything production-premium.

## What it does
- **gsap-core** — core API (gsap.to/from/fromTo), easing, duration, stagger, defaults, matchMedia for responsive + prefers-reduced-motion.
- **gsap-timeline** — sequencing, position parameter, nesting, playback control.
- **gsap-scrolltrigger** — scroll-linked animation, pinning, scrub, parallax triggers.
- **gsap-plugins** — ScrollTo, ScrollSmoother, Flip, Draggable, Inertia, Observer, SplitText, ScrambleText, SVG/physics, CustomEase.
- **gsap-react** — GSAP inside React (useGSAP, refs, cleanup/scoping).
- **gsap-frameworks** — GSAP in Vue/Nuxt/Svelte (lifecycle, scoped selectors, cleanup).
- **gsap-utils** — clamp, mapRange, normalize, interpolate, random, snap, toArray, wrap, pipe.
- **gsap-performance** — prefer transforms, avoid layout thrashing, will-change, batching for 60fps.
