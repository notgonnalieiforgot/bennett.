# Image Generation & Visual Direction

The asset-and-reference layer: skills that generate design imagery to direct a build, or to ship as finished creative. Reach for these to set art direction before coding, or to produce the visuals a site/campaign needs.

## What it does
- **imagegen-frontend-web** — generates one horizontal design reference image per section for web layouts, with a consistent palette and varied composition.
- **imagegen-frontend-mobile** — premium mobile app screen concepts framed in phone mockups; images only, no code.
- **image-to-code-skill** — generates design images first, analyzes them, then implements the site to match them closely.
- **canvas-design** — original visual art as PNG/PDF (posters, designs) built from design philosophy.
- **banner-design** — social/ad/web-hero/print banners with AI-generated visuals across multiple art directions.

## Used in / pairs with
- [[Higgsfield]] — the AI image/video generator that produces art-directed plates and per-section assets.
