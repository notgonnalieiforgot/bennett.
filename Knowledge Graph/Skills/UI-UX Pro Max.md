# UI-UX Pro Max

A design-intelligence skill: a deep library of styles, color palettes, font pairings, product-type patterns, UX guidelines, and chart types spanning many frontend stacks. Reach for it to plan, build, review, or improve UI/UX with concrete, opinionated design knowledge rather than guesses.

## What it does
- Ships 50+ visual styles (glassmorphism, brutalism, minimalism, neumorphism, bento grid, dark mode, claymorphism, flat, skeuomorphism, and more).
- 160+ color palettes, 50+ font pairings, 160+ product-type patterns, ~100 UX guidelines, and 25 chart types.
- Works across React, Next.js, Vue, Svelte, SwiftUI, React Native, Flutter, Tailwind, shadcn/ui, and plain HTML/CSS.
- Actions span the lifecycle — plan, build, create, design, implement, review, fix, improve, optimize, and check UI/UX code — for websites, landing pages, dashboards, e-commerce, SaaS, portfolios, and mobile apps.
- Integrates the shadcn/ui MCP for component search and examples.
