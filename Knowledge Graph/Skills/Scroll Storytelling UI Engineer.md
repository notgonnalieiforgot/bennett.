# Scroll Storytelling UI Engineer

End-to-end scroll storytelling engineering: GSAP ScrollTrigger pinning/scrub/stacking, performance tuning (INP/CLS), editorial typography, Framer micro-interactions, loading screens, and exploded 3D scrollytelling. Reach for it for parallax, pinned sections, scrub timelines, product-explode effects, or any UI motion audit.

## What it does
- Implements GSAP ScrollTrigger patterns: pinned sections, scrubbed timelines, sticky stacking layers, and single-timeline orchestration.
- Treats performance as a first-class concern — keeps INP and CLS low so premium motion does not tank Core Web Vitals.
- Covers supporting craft: wide editorial typography, Framer Motion micro-interactions, loading screens, and exploded 3D scrollytelling sequences.
- Runs motion audits on existing sites to find jank, layout shift, and degraded mobile behavior.
