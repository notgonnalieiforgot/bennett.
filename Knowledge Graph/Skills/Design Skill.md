# Design Skill

A comprehensive brand-and-asset design skill: brand identity, design tokens, logos, corporate identity programs, banners, icons, HTML presentations, and social photos. Reach for it when a project needs visual identity work and marketing assets, not just interface code.

## What it does
- **Logos** — 55 styles, generated with Gemini AI.
- **Corporate Identity Program (CIP)** — up to 50 deliverables plus CIP mockups.
- **Banners** — 22 styles across social, ads, web heroes, and print (Facebook, X, LinkedIn, YouTube, Instagram, Pinterest, TikTok, Threads, Google Ads).
- **Icons** — 15 styles, SVG output via Gemini.
- **Social photos** — HTML→screenshot pipeline sized per platform.
- **Design tokens, UI styling, and HTML presentations** (Chart.js) round out the kit for full brand systems.
