# Frontend Design & Taste Skills

A cluster of design-quality skills (frontend-design, taste-skill, soft-skill, impeccable, redesign-skill) that produce distinctive, production-grade UI and actively block generic "AI default" aesthetics. Reach for them whenever a build needs to look intentional, expensive, and hand-crafted rather than templated.

## What it does
- **frontend-design** — creates distinctive, polished frontend code and UI that avoids generic AI looks; for pages, components, artifacts, and beautifying any web UI.
- **taste-skill / soft-skill** — senior UI/UX engineering with metric-based rules, strict component architecture, hardware-accelerated CSS, and the exact fonts/spacing/shadows that make a site feel high-end; blocks cheap defaults.
- **impeccable** — design, redesign, audit, polish, animate, and harden interfaces; UX review, hierarchy, accessibility, motion, and reusable tokens.
- **redesign-skill** — upgrades existing sites/apps to premium quality, auditing for generic patterns without breaking functionality.
