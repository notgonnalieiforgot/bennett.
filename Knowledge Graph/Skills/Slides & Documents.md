# Slides & Documents

Strategic document and presentation production: HTML decks with Chart.js and design tokens, and polished real-estate booklets exported as .pptx. Reach for it when a deliverable is a document or presentation rather than a deployed web interface.

## What it does
- **slides** — strategic HTML presentations with Chart.js, design tokens, responsive layouts, and copywriting formulas for pitches, proposals, and client reports.
- **luxury-home-booklet** — luxury spec-home introduction booklets as polished .pptx files for real-estate marketing and property launches.

- [[Projected-MRR Handoff Docs]] — the slides skill is the production layer for the MRR Google Docs handed to clients at project close.
- [[Project Handoff Playbook]] — final handoff packages typically include a polished deck alongside the live site and audit.
- [[Image Generation & Visual Direction]] — generated imagery and Higgsfield visuals populate hero slides and booklet spreads.
