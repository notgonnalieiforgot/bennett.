# Visual Style Skills

A set of opinionated aesthetic engines, each committing to a specific visual language so a build reads as a deliberate art direction instead of a default theme. Pick one to anchor a project's look, then enforce it with the taste skills.

## What it does
- **brutalist-skill** — raw mechanical Swiss-print + military-terminal aesthetic; rigid grids, extreme type-scale contrast, analog degradation.
- **minimalist-skill** — clean editorial UI; warm monochrome, typographic contrast, flat bento, no gradients or heavy shadows.
- **gpt-tasteskill** — GSAP-heavy taste engine enforcing Python-driven layout randomization, AIDA structure, gapless bento, and strict ScrollTriggers.
- **design-taste-frontend** — senior UI/UX engineering with metric-based rules, strict component architecture, and hardware acceleration.
- **stitch-skill** — generates DESIGN.md files for Google Stitch that enforce anti-generic, premium UI standards.
- **theme-factory** — 10 preset themes plus on-the-fly theme generation applied to any artifact (slides, docs, landing pages).
